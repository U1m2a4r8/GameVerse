# GameVerse

A full-stack gaming platform built with the MERN stack (MongoDB, Express, React, Node.js), deployed to AWS with a DevOps-driven pipeline.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Infrastructure:** AWS, CI/CD tooling

## Status
🚧 Repository scaffold — core application code is being migrated in.

## Setup
```bash
npm install
npm run dev
```
